    1. W3C
      World Wide Web Consortium, una organización que desarrolla estándares web.
    2. WHATWG
      Web Hypertext Application Technology Working Group, organización enfocada en el desarrollo de HTML y APIs del navegador.
    3. HTML
      Lenguaje de marcado utilizado para estructurar y presentar contenido en la web.
    4. XHTML
      Extensible HyperText Markup Language, una reformulación de HTML como aplicación XML.
    5. DTD
      Definición de tipo de documento, especifica las reglas y estructuras válidas en un documento XML o HTML.
    6. APIs
      1. Drag and Drop
        API para arrastrar y soltar elementos en la interfaz de usuario.
      2. Geolocalización
        API que permite obtener la ubicación geográfica del dispositivo del usuario.
      3. Almacenamiento local
        API para almacenar datos localmente en el navegador del usuario.
      4. App Cache
        API que permite almacenar una aplicación web en el caché del navegador para funcionar sin conexión.
      5. WebWorkers
        API que permite ejecutar scripts en segundo plano para mejorar el rendimiento.
    7. Editor WYSIWYG
      Un editor visual que permite crear contenido sin necesidad de escribir el código HTML manualmente.

      1. Mozilla Firefox
        Navegador web desarrollado por Mozilla.

        Navegador web desarrollado por Google.
      3. Safari
        Navegador web desarrollado por Apple.
