Introducción a HTML5

HTML5 es una revisión importante del lenguaje de marcado HTML (HyperText Markup Language), que define la estructura y el contenido de las páginas web.

Características de HTML5

Algunas de las principales características de HTML5 son:

  • Mejora del soporte multimedia
    • Audio
    • Video
  • Nuevos elementos semánticos
    • article
    • section
    • footer

Logotipo de HTML5

Para más información sobre HTML5, visita la página de Wikipedia sobre HTML5.

Comparación de versiones de HTML

Versión Año de lanzamiento Características clave
HTML4 1997 Introducción de hojas de estilo (CSS)
XHTML 2000 Más estricto y basado en XML
HTML5 2014 Soporte para multimedia y gráficos

HTML5 es un estándar ampliamente utilizado.
Aprenderlo es muy útil para el desarrollo web moderno.
