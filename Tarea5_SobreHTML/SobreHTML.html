Explicación de HTML, XHTML y organismos relacionados

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado fundamental para crear y estructurar páginas web. Utiliza etiquetas para organizar el contenido, permitiendo a los navegadores mostrar textos, imágenes, enlaces, formularios y otros elementos en una página web.

Relación de Tim Berners-Lee con HTML

Tim Berners-Lee, un científico de computación, creó HTML a finales de la década de 1980, al desarrollar la primera versión de este lenguaje. Su propósito era compartir información entre investigadores en la Web y, gracias a su trabajo, se convirtió en el padre de la World Wide Web.

Versiones de HTML

  • HTML 1.0 - La primera versión, publicada en 1991.
  • HTML 2.0 - Estándar formalizado en 1995.
  • HTML 3.2 - Publicado en 1997, con nuevas características como tablas y scripts.
  • HTML 4.01 - Versión revisada en 1999, muy utilizada durante años.
  • HTML5 - Publicado en 2014, con mejoras para multimedia y aplicaciones web.

¿Qué es la W3C?

La W3C (World Wide Web Consortium) es una organización internacional que desarrolla estándares para la Web, como HTML, CSS y otros, con el fin de promover una mayor accesibilidad y funcionalidad.

¿Qué es el WHATWG?

El WHATWG (Web Hypertext Application Technology Working Group) es un grupo de desarrolladores que trabaja en el mantenimiento de HTML como un estándar vivo (Living Standard), enfocándose en la evolución constante del lenguaje para adaptarse a las necesidades modernas de la web.

¿Qué es un "Living Standard"?

Un Living Standard es un estándar que se actualiza de manera continua, en lugar de tener versiones fijas. HTML, desde HTML5, es considerado un estándar vivo, lo que significa que se actualiza constantemente según las necesidades tecnológicas.

¿Qué es XHTML?

XHTML (eXtensible HyperText Markup Language) es una versión de HTML basada en XML, diseñada para mejorar la interoperabilidad y la consistencia en los documentos web. XHTML exige reglas de sintaxis más estrictas, por lo que el contenido debe estar bien estructurado.

Versiones de XHTML

  • XHTML 1.0 - Basado en HTML 4.01, publicado en 2000.
  • XHTML 1.1 - Publicado en 2001, con mejoras en modularidad.
  • XHTML 2.0 - Un intento no exitoso de evolución de XHTML.

Diferencias entre XHTML y HTML

Las principales diferencias entre XHTML y HTML incluyen:

  • Sintaxis estricta: En XHTML, todas las etiquetas deben estar correctamente anidadas y cerradas, mientras que en HTML no siempre es obligatorio.
  • Mayúsculas y minúsculas: XHTML es sensible a mayúsculas, por lo que las etiquetas deben estar en minúsculas.
  • Atributos obligatorios: En XHTML, los atributos deben tener valores en comillas, a diferencia de HTML, donde algunos atributos pueden no tener comillas.
